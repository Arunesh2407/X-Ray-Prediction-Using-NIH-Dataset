# Retrieval Data

Expected JSON structure:

```json
{
  "Pneumonia": [
    {
      "source": "Radiopaedia",
      "title": "Example title",
      "snippet": "Evidence text for retrieval.",
      "url": "https://example.org",
      "score": 0.9
    }
  ]
}
```

You can also store a list of labeled snippet objects with a `label` field.
